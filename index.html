Operator Logika

Operator logika adalah operator yang digunakan untuk melakukan operasi logis pada nilai boolean (true/false).

macam macam operator logika adalah :

  1. And ( && )
  2. Or ( || )
  3. not ( ! )
  4. dst

Operator Aritmatika

Operator aritmatika digunakan untuk melakukan operasi matematika seperti penjumlahan, pengurangan, perkalian, dan sebagainya.

macam-macam operator aritmatika adalah :

  1. penjumlahan (+)
  2. pengurangan (-)
  3. pembagian (/)
  4. perkalian (*)
  5. bagi sisa (%)
  6. kuadrat (**)
  7. dst
